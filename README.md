## 🎯 Sobre el Proyecto

**Crypto E-Duty** es una plataforma educativa **open source** diseñada para democratizar el acceso al conocimiento sobre criptomonedas, blockchain y finanzas descentralizadas. 

### 💡 ¿Por qué existe este proyecto?

En el siglo 21, las criptomonedas y la tecnología blockchain están transformando la forma en que interactuamos con el dinero y los activos digitales. Sin embargo, el ecosistema puede ser abrumador para los recién llegados, con:

- **Términos técnicos complejos**
- **Riesgos de seguridad** (estafas, phishing, pérdida de fondos)
- **Falta de recursos educativos accesibles**
- **Barreras de entrada** (costos, complejidad técnica)

**Crypto E-Duty** resuelve estos problemas ofreciendo:

✅ **Aprendizaje práctico** con fondos ficticios (sin riesgo)
✅ **Contenido claro y estructurado** para todas las edades
✅ **Interfaz profesional y accesible**
✅ **Código abierto** para transparencia y colaboración
✅ **Totalmente gratuito** y sin publicidad

### 🎓 Utilidad para Entornos Educativos

Esta plataforma es ideal para:

- **Escuelas y universidades** que quieren enseñar sobre tecnología blockchain
- **Talleres y cursos** sobre criptomonedas
- **Autoaprendizaje** para personas interesadas en el mundo cripto
- **Demostraciones técnicas** de conceptos blockchain
- **Proyectos comunitarios** de educación financiera

### 🔓 ¿Por qué Open Source?

Ser open source significa:

🔹 **Transparencia**: Cualquiera puede auditar el código
🔹 **Colaboración**: La comunidad puede contribuir y mejorar
🔹 **Accesibilidad**: Gratis para todos, sin barreras
🔹 **Educación**: Los estudiantes pueden aprender del código
🔹 **Confianza**: No hay código oculto o malicioso

## 🚀 Comenzar

### Requisitos

- [Node.js](https://nodejs.org/) 18.0.0 o superior
- [npm](https://www.npmjs.com/) 9.0.0 o superior
- [Git](https://git-scm.com/) (opcional, para despliegue)

### Instalación

1. **Clona el repositorio:**

```bash
git clone https://github.com/TU_USUARIO/crypto-e-duty.git
cd crypto-e-duty
```

2. **Instala las dependencias:**

```bash
npm install
```

### Desarrollo

Inicia el servidor de desarrollo:

```bash
npm run dev
```

- La aplicación estará disponible en: [http://localhost:3000](http://localhost:3000)
- Se recargará automáticamente cuando hagas cambios
- Puedes ver errores en la consola

### Construcción

Genera los archivos para producción:

```bash
npm run build
```

- Los archivos se generarán en la carpeta `dist/`
- Optimizados para producción
- Listos para desplegar

### Despliegue en GitHub Pages

1. **Configura el repositorio:**

   - Asegúrate de que `package.json` tenga el `homepage` correcto:
     ```json
     "homepage": "https://TU_USUARIO.github.io/crypto-e-duty/"
     ```

   - Asegúrate de que `vite.config.js` tenga el `base` correcto:
     ```javascript
     base: '/crypto-e-duty/'
     ```

2. **Despliega:**

```bash
npm run deploy
```

3. **Configura GitHub Pages:**

   - Ve a **Settings > Pages** en tu repositorio
   - Selecciona la rama: `gh-pages`
   - Carpeta: `/ (root)`
   - Haz clic en **Save**

4. **¡Listo!** 🎉

   Tu sitio estará disponible en:
   [https://juansinjcy.github.io/cryptoeduty_project/](https://juansinjcy.github.io/cryptoeduty_project/)

## 📖 Ruta de Aprendizaje

La plataforma ofrece **10 módulos** estructurados en 4 niveles:

### 🟢 Nivel 1: Principiante

1. **Introducción a Criptomonedas** (15 min)
   - ¿Qué son las criptomonedas?
   - Diferencias con el dinero tradicional
   - Beneficios y riesgos
   - Casos de uso reales

2. **Blockchain Básico** (20 min)
   - ¿Qué es blockchain?
   - Validación de transacciones
   - Proof of Work vs Proof of Stake
   - Inmutabilidad y transparencia

### 🟡 Nivel 2: Intermedio

3. **Wallets y Almacenamiento** (25 min)
   - Tipos de wallets
   - Claves públicas y privadas
   - Frases semilla
   - Seguridad básica

4. **Transacciones y Direcciones** (20 min)
   - Anatomía de una transacción
   - Comisiones de red
   - Tiempo de confirmación
   - Exploradores de blockchain

5. **Seguridad en Cripto** (30 min)
   - Estafas comunes
   - Phishing y ingeniería social
   - Protección contra malware
   - Backup de wallets

### 🔵 Nivel 3: Avanzado

6. **Bitcoin en Profundidad** (35 min)
   - Historia de Bitcoin
   - Minería y nodos
   - Halving y oferta limitada
   - Lightning Network

7. **Ethereum y Smart Contracts** (40 min)
   - ¿Qué es Ethereum?
   - Smart contracts
   - Gas y comisiones
   - Tokens ERC-20 y ERC-721

8. **Finanzas Descentralizadas (DeFi)** (45 min)
   - ¿Qué es DeFi?
   - Exchanges descentralizados
   - Préstamos y yield farming
   - Riesgos en DeFi

9. **NFTs y Tokens No Fungibles** (30 min)
   - ¿Qué es un NFT?
   - Casos de uso reales
   - Estándares de tokens
   - Mercados de NFTs

### 🜨 Nivel 4: Experto

10. **El Futuro de Cripto** (30 min)
    - Web3 y la internet descentralizada
    - DAOs (Organizaciones Autónomas)
    - Metaverso y cripto
    - Tecnologías emergentes

### 🎉 Certificado de Finalización

Al completar todos los módulos:
- Obtienes un **certificado virtual** de finalización
- Puedes descargar la **guía completa en PDF**
- Desbloqueas **contenido adicional**

1. Abre un [Issue](https://github.com/juansinjcy/crypto-e-duty/issues)
2. Describe el problema con detalle
3. Incluye pasos para reproducirlo

2. Describe tu idea
3. Explica el beneficio
4. Incluye ejemplos si es posible

### 🔧 Enviar Pull Requests

1. **Fork** el repositorio
2. Crea una **rama** para tu funcionalidad (`git checkout -b feature/nueva-funcionalidad`)
3. **Commit** tus cambios (`git commit -m 'Añade nueva funcionalidad'`)

5. Abre un **Pull Request**
